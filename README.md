## 🌐 Сайт-портфолио

**Демо:** [https://pickus228-code.github.io/portfolio-n-tea/](https://pickus228-code.github.io/portfolio-n-tea/)

Одностраничный сайт-визитка для чайного ларька в Минске. Сочетает в себе функционал каталога, формы заказа и личного портфолио разработчика.

---

## 📋 О проекте

Этот проект — портфолио-визитка, созданная для демонстрации навыков веб-разработки. Сайт сочетает в себе:

*   **Каталог чая:** Список из 8 сортов с ценами за 100г.
*   **Форма заказа:** Выбор чая и количества, автоматический подсчет суммы.
*   **Контакты:** Адрес (пр. Независимости, 68), время работы и способ связи.
*   **Портфолио разработчика:** Блок с информацией о Дианазавр (возраст, хобби, отзыв о курсах).
*   **Интерактив:** Тёмная тема, смена цвета обводки, загрузка аватара, модальные окна для входа/регистрации и записи на курсы.

---

## 🛠️ Технологии

*   **HTML5** — Семантическая разметка.
*   **CSS3** — Кастомные стили, адаптивная верстка (Flexbox/Grid), анимации, поддержка тёмной темы.
*   **JavaScript (Vanilla)** — Вся логика: заказ, модальные окна, переключение тем, смена аватара и цвета.
*   **FontAwesome** — Иконки.
*   **GitHub Pages** — Хостинг.

## ⚙️ Основные функции

| Функция | Описание |
|---------|----------|
| 🛒 **Заказ чая** | Выбор сорта и количества, мгновенный подсчет стоимости. |
| 🍃 **Каталог** | Отображение всех доступных видов чая с ценами. |
| 📍 **Адрес и контакты** | Реальные данные чайного ларька в Минске. |
| 🎨 **Смена темы** | Переключение между светлой и тёмной темой. |
| 🎨 **Смена цвета обводки** | Выбор пастельного цвета для рамок элементов. |
| 👤 **Смена аватара** | Загрузка собственного фото в профиль. |
| 🔐 **Модальные окна** | Вход/регистрация, запись на курсы. |
| 🎵 **Блок с музыкой** | Ссылки на авторские треки о чае. |

## 📱 Адаптивность

Сайт корректно отображается на устройствах с разным размером экрана:
*   Десктопы (ширина > 900px) — двухколоночная сетка.
*   Планшеты (900px – 480px) — одноколоночная сетка, увеличенные элементы.
*   Мобильные устройства (< 480px) — оптимизированные отступы, шрифты и размеры кнопок.
